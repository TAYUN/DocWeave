# P08 设置页

## 路由

`/settings/*`

## 用途

集中承载个人设置、工作区设置和后续系统级配置入口。

## 当前阶段定位

- 当前为占位规格
- 作为后续阶段页面保留索引与导航锚点
- 当前代码库中没有对应已落地页面，本文件不代表当前 UI 已存在

## 最小规格

- 主任务：查看和修改账户或工作区相关配置
- 主内容：设置分组导航、设置表单、保存反馈
- 状态要求：加载中、保存中、保存成功、保存失败、disabled、restricted / 无权限
- 导航关系：从用户菜单或工作区菜单进入
- 实现边界：保持普通工作台结构，不突然切成品牌化后台页

## 状态要求细化

- `disabled`：当前项不可编辑时，控件保持可见并说明受限原因，例如只读配置或缺少前置条件
- `restricted / 无权限`：无权访问某个设置分区时，页面或分区需要明确说明权限限制，并提供返回上一级设置或工作台的路径

## 后续补全时必须明确

- 个人设置与工作区设置是否拆子路由
- 权限模型如何影响可见项与可编辑项
- 是否需要单独的 AI 配置分区
