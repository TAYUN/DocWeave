# 47. 视觉品牌资产与图标映射清单

阅读说明：

1. 本文档负责把品牌标识与高频功能图标做成最小映射清单。
2. 图标与视觉标识基线以 [25. 图标与视觉标识方案](../decisions/25.%20图标与视觉标识方案.md) 为准。

## 1. 文档目的

本文档用于给设计与实现阶段一个最小共享参考，避免同一语义在不同页面用不同图标，或品牌标识在登录页、工作台、分享页三套表达。

## 2. 品牌标识建议落点

建议至少区分：

1. 主 Logo：登录页、公开页
2. 简化 Mark：侧边栏折叠态、favicon
3. Wordmark：品牌介绍区域

## 3. 高频功能语义建议

建议优先统一以下语义：

1. 文档
2. 空间
3. 搜索
4. AI / 聊天
5. 分享
6. 版本历史
7. 设置
8. 删除 / 归档

## 4. 当前结论

这份清单的作用不是替代完整设计系统，而是先把最容易散掉的品牌与功能图标语义统一起来，减少实现阶段来回重改。
