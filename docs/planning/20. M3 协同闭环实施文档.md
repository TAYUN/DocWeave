# 20. M3 协同闭环实施文档

阅读说明：

1. 本文档负责把 M3 协同闭环目标转成可执行实施范围。
2. 总路线以 [00. 当前实施路线图](./00.%20当前实施路线图.md) 为准。
3. 协同协议以 [16. 协同 Token 与 Presence 设计](../architecture/16.%20协同%20Token%20与%20Presence%20设计.md) 为准。

## 1. 阶段目标

让两个浏览器可以同时编辑同一篇文档，并看到基础在线状态。

## 2. 本阶段包含

1. `apps/collab` 独立启动。
2. 文档房间接入。
3. 协同 token 签发与校验。
4. 正文同步。
5. 基础 Presence 展示。

## 3. 本阶段不包含

1. 协同持久化恢复。
2. 评论、批注等更多实时对象。
3. 复杂冲突审计。

## 4. 最小任务

1. 落地房间命名与 token contract。
2. `apps/api` 提供 `POST /api/collaboration/token`。
3. `apps/collab` 校验 token 并接受连接。
4. `apps/web` 完成 `Yjs + Hocuspocus` 接入。
5. 基础在线成员展示可用。

## 5. 完成标准

1. 两个浏览器可协同编辑同一文档。
2. 标题与正文同步一致。
3. 在线成员展示成立。
4. 无权限用户无法获取协同 token。
